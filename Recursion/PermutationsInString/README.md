# Permutations In String

**Source**: [https://leetcode.com/problems/permutation-in-string/](https://leetcode.com/problems/permutation-in-string/)

## Problem Description

Given two strings s1 and s2, write a function to return true if s2 contains the 
permutation of s1. In other words, one 
of the first string's permutations is the substring of the second string.
