# Valid Parentheses

**Source**: [https://leetcode.com/problems/valid-parentheses/](https://leetcode.com/problems/valid-parentheses/)

## Problem Description

Given a string `s` containing just the characters `(`, `)`, `{`, `}`, `[`, `]`, determine if the input string is valid. Brackets must be closed in the correct order and by the same type.

**Example**: `"()[]{}"` → `True`, `"(]"` → `False`
